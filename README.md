# Java Commandline Calculator

## How-Tos

### Build

```
gradlew bootJar
```

### Run

Windows: use "PowerShell" or "Command Prompt". "Git Bash(mintty)" is not supported.
macOS, Linux: use terminal.

```
java -jar .\build\libs\commandline-calculator-0.0.1-SNAPSHOT.jar
```
